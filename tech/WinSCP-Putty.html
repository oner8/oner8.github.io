Linux管理利器 WinSCP 设置中文与自动关联 Putty

发布于 2017-08-17 tech  WinSCP 设置中文 Putty WinSCP关联Putty 

WinSCP 是一个 Windows 环境下使用的 SSH 的开源图形化 SFTP 客户端。它的主要功能是在本地与远程计算机间安全地复制文件，并且可以直接编辑文件。而Putty则是一款开源、小巧的SSH工具，两者结合，堪称完美。

在这里我不得不吐槽下一直使用的Xshell工具，最近爆出后门，对是后门不是漏洞，连官方下载的都中招，更别说第三方网站了，真TM操碎了心。此文所有链接提供官方地址，也推荐这样的方法，相对更加安全。

下载WinSCP

官方下载地址：WinSCP下载

WinSCP支持绿色版和安装版，请自行通过上面的地址下载并安装。

设置中文

启动WinSCP发现是英文界面，博主这种英语渣不太方便操作，官方有提供中文语言包，访问：WinSCP语言包，找到Simplified Chinese并下载。

将语言包解压放到WinSCP安装目录，并重启WinSCP，必须先连接一个主机才能打开设置，输入快捷键Ctrl + Alt + P打开如下界面，设置为简体中文并重启WinSCP就能看到中文界面了。

关联Putty

WinSCP关联Putty后可以快速通过Putty打开SSH终端，输入快捷键Ctrl + Alt + P打开设置 – 集成 – 应用程序 – 设置Putty客户端路径，如下截图。

关联Putty后，输入快捷键Ctrl + P就可以快速打开Putty，或者点击下图中的按钮。

总结

WinSCP用来上传下载文件，Putty连接SSH终端管理服务器，两者结合非常完美。第三方网站也提供了诸如WinSCP xx中文版软件，但博主非常不推荐，此类软件还是去官方下载相对安全一些。
