Bing 每日一图 amp 随机图片 API

发布于 2017-08-28 tech  Bing 每日一图 Bing 每日一图API 

大家都知道微软的Bing搜索引擎首页每天都会提供了一些有趣的图片，而这些图片很多都是有故事含义的，很多网友每天去访问bing首页都是为了这些图片而去的。

本人提供bing每日一图接口：

右键另存为可下载bing高清图片

使用方法：

https://api.wrdan.com/randimg

这个API接口可以直接把它当做一个链接来用，插入如下代码：

<img src="https://api.wrdan.com/randimg" alt="Bing每日图片" />
