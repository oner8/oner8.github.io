推荐几款非常不错的免费 SSL

 2016-12-11 682 2 tech  免费ssl Let’s Encrypt SSL StartSSL 

简介

传输层安全协议（英语：Transport Layer Security，缩写：TLS），及其前身安全套接层（Secure Sockets Layer，缩写：SSL）是一种安全协议，目的是为互联网通信，提供安全及数据完整性保障。网景公司（Netscape）在1994年推出首版网页浏览器，网景导航者时，推出HTTPS协议，以SSL进行加密，这是SSL的起源。IETF将SSL进行标准化，1999年公布了第一版TLS标准文件。随后又公布了 RFC 5246 （2008年8月）与 RFC 6176 （2011年3月）。在浏览器、电子邮件、即时通信、VoIP、网络传真等应用程序中，广泛支持这个协议。主要的网站，如Google、Facebook等也以这个协议来创建安全连接，发送数据。目前已成为互联网上保密通讯的工业标准。（转自：维基百科）

免费SSL

现在免费的SSL 越来越多，接入SSL的成本也原来越低。

有哪些免费的SSL呢？

Let’s Encrypt SSL

项目主页：https://github.com/letsencrypt/letsencrypt

国外一个公共的免费SSL项目。Let’s Encrypt免费SSL证书有效期是90天，也就是每三个月你就得续期一次。

网上有好多续期的脚本，自己善于搜索吧。

腾讯云DV SSL

申请地址：https://console.qcloud.com/ssl

腾讯云免费提供一年的DV SSL。 目前不知道如何延期以及延期收费情况。

AlphaSSL证书

申请地址：https://portal.loovit.net/cart.php?a=add&pid=1

国人搞的一个： https://alpha.80host.com/

优点支持泛域名，缺点据说是野卡，可能会被吊销。不过吊销的成本也挺高的，没那么容易吊销吧。

阿里云 DV SSL

申请地址： https://common-buy.aliyun.com/?spm=5176.7968328.231195.3.DAY7Db&commodityCode=cas#/buy

阿里云提供的是 Symantec 的证书。

自己签发SSL证书

其实，SSL证书是可以自己签发的。缺点是，浏览器必须导入自己证书。如果让朋友用，朋友的浏览器也需要导入证书。

StartSSL

申请地址：https://startssl.com

Class 1, Class 2, Class 3, Extended Validation, 对于申请者的验证严格程序依次递增, 同时安全性也随之增强.Class 1 属于 免费的类型。

谨慎申请使用这家的SSL

WoSign(沃通)

申请地址： https://buy.wosign.com/free

这家一直支持免费的证书，偶尔有活动能签发 100+域名。

兼容性略差一点。

谨慎申请使用这家的SSL

后续更新

  • Mozilla表态将停止信任WoSign 和StartCom签发的新证书，已签发的证书不受影响。
  • 新增 阿里云Symantec DV SSL。
  • WoSign决定暂时关闭免费SSL证书的申请。2016年9月29日。
