如何获取 Chrome 插件的下载地址

发布于 2017-08-24 tech 

使用过 Chrome 浏览器的用户都应该安装过插件，但是我们从 Google 的应用商店下载插件是无法直接获取到下载地址的。

不过我们总是有些需求需要获取到这些插件的地址，比如朋友想安装某个插件，但是因为某些原因无法访问 Google 应用商店，而我可以访问，这时候我们就想如果能获取到插件的下载地址，直接下载好然后发送给朋友多好啊。这篇文章就教大家如何获取 Chrome 插件的下载地址。

这里以我们最常使用的 Adblock plus 插件为例，一步一步教教大家获取 Adblock plus 插件的下载地址。

首先，我们访问 Google 应用商店插件下载页面时，其 URL 上面会有一个id，如下图红色框圈住的一串字母（cfhdojbkjhnklbpkdaibdccddilifddb），这个就是 Adblock plus 插件的id。（ Adblock plus 插件下载页为：https://chrome.google.com/webstore/detail/adblock-plus/cfhdojbkjhnklbpkdaibdccddilifddb?hl=zh-CN）

其次，所有 Chrome 插件可以通过下面URL下载：

https://clients2.google.com/service/update2/crx?response=redirect&prodversion=49.0&x=id%3D~~~~%26installsource%3Dondemand%26uc

上面 URL 中四个波浪号(~)就是插件的 id，现在我们将 Adblock plus 插件的 id （cfhdojbkjhnklbpkdaibdccddilifddb）替换掉它，得到下面 URL：

https://clients2.google.com/service/update2/crx?response=redirect&prodversion=49.0&x=id%3Dcfhdojbkjhnklbpkdaibdccddilifddb%26installsource%3Dondemand%26uc

好了，到这里你已经得到 Chrome 插件的下载地址了。秘籍就是获取到插件的 id ，然后替换上面四个波浪号(~)就可以啦，很简答。下载的插件名称是 extension_[version].crx，其中 version 是咱们下载插件的最新版。所有如果我们使用上面 URL 下载 Adblock plus 插件，下载之后插件名称将会是extension_1.13.3.crx。因为截至本文撰写时，最新版的 Adblock plus 插件就是1.13.3。

需要注意的是，上面的 URL 需要翻墙才能访问。
