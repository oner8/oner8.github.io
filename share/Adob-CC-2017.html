版本特点及安装注意事项：

1：系统需求：Win7 64位及以上。
2：本版仍为传统模式，不含ACCC，也不接受ACCC安装启动卸载。ACCC 启用 packageCode 管理新版程序（即HD），抛弃了 productCode，故不显示已安装。
3: 本版集成 Adobe Acrobat DC 特别版，
集成新版AMTLIB模拟破解(v0.9.2)。
集成最新版AAMv10(CS10)/ACCC SP。
移除试用提示。
移除自动更新，移除菜单更新。
打开、保存文件时不显示在线存储。
启动欢迎页不显示资讯。
最近使用列表中的最大文档数量默认为零。
4：Adobe Lightroom 强制为订阅版。
5：Illustrator CC 2018 原版已修正中文3D凸出和斜角选项乱码的问题。
6：ExtendScript Toolkit 和 Captivate 的简体中文/繁体中文语言包均由本人独立汉化完成。
原生模式，不影响其他语言的安装使用。禁止重打包发布。
7: CC 2018 支援各类新硬件、新特性，对硬件有较高的要求。极不推荐老旧设备上跑新版本，主观上的使用感受会不如旧版本。
内存建议8GB、推荐16GB及以上。
显存建议2GB、推荐3GB及以上。
硬盘建议固态硬盘、部分程序（如AE、PR）建议配备NVME固态硬盘（推荐三星SM951、PM951及以上）
请务必理解，处理复杂文件需要较高的配置才能满足软件的内存和高速缓存请求、保证软件的响应速度。
建议1080P分辨率以上显示设备、推荐配备数位板/触摸屏。
注：新版本支持GPU硬件加速的显卡列表中已没有2010年之前的显卡。
台式机双显卡：必须确保为官方最新驱动，在设置中选择使用独立显卡启动软件。
笔记本双显卡：必须确保为官方最新驱动。笔记本双显驱动支持年限较短，旧版双显卡驱动是不支持GPU硬件加速的。
NVIDIA显卡：在出现迷之BUG时，请首先考虑在软件中关闭GPU硬件加速。
AMD显卡：建议 Win10 64位及以上系统。
若没有其他特别需求，Adobe程序推荐AMD显卡。

安装流程：

下载压缩包后建议校验。
WinRAR 5.0以上版本解压；用虚拟光驱加载ISO，不要解压ISO。
本版分为大师版和独立版，可根据个人需要安装。
大师版、独立版均为一键安装，一般无需任何操作。建议以管理员权限安装。
————————————————————————-
解包而不使用镜像安装请务必注意不要将安装程序放置在过长的路径下导致的安装失败！
————————————————————————————-

注意：关于VC运行库问题

Microsoft Visual C++ 2005、2008、2010、2012、2013、2015 Redistributable 32位、64位
针对很多非纯净系统环境下在安装过程中出现的VC运行库无法安装导致安装失败、安装程序错误等问题，
建议先检查、卸载本地所有（2005、2008、2010、2012、2013、2015）各种版本的VC运行库，
手动安装 .\Content\Runtime\ 目录下所有VC运行库后，再尝试安装程序。
或者安装各类VC运行库合集包避免此类问题。

————————————————-

【如果在安装前需要清理残留的安装、激活信息】

一、使用官方清理工具先清理已安装程序
.\Content\Tools\Adobe Creative Cloud Cleaner Tool.exe
二、清理激活信息（删除文件、文件夹）
C:\ProgramData\Adobe\SLStore
C:\Program Files (x86)\Common Files\Adobe\SLCache
C:\ProgramData\regid.1986-12.com.adobe
三、清理残留安装信息（删除文件、文件夹）
C:\Program Files (x86)\Common Files\Adobe\backup
C:\Program Files (x86)\Common Files\Adobe\caps
C:\Program Files (x86)\Common Files\Adobe\Installers
四、检查本机AAM版本。Acrobat 许可模块安装失败，正版软件验证失败等问题请直接删除AAM文件目录。
（请注意查找并终止此目录下正在运行的进程，如AdobeIPCBroker.exe。原版 ACCC 桌面请先卸载。
已安装特别版 Adobe Creative Cloud 桌面，请同时终止Adobe Desktop Service.exe和其子进程）
C:\Program Files (x86)\Common Files\Adobe\OOBE\

———————————————————

本版使用的是DLL PATCH + EMU 混合处理。默认全局屏蔽自动更新。
新版本安装、更新模式已变更，由ACCC完全接管操作，想用ACCC自动更新的请购买订阅。

手动更新：等待本特别版的更新，卸载旧版，安装新版即可。
