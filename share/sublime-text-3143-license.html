如果注册后过一段时间出现“Your license key is no longer valid, and has been removed”激活错误，请修改host文件以屏蔽官方验证。

LINUX系统下在终端输入命令sudo vim /etc/hosts，打开hosts文件，最后面加上

# Sublime Text 3

0.0.0.0 license.sublimehq.com
0.0.0.0 45.55.255.55
0.0.0.0 45.55.41.223

然后按esc键，输入:wq保存（有冒号）。最后sudo /etc/init.d/networking restart重新载入文件就好了。

WINDOWS应该一样，把那段话加到hosts文件后面然后重启应该就行了。
